# Video 01-01 — Preparación del entorno de trabajo

## Objetivo de aprendizaje

Conocer y configurar los requisitos mínimos para seguir el curso: VS Code, una cuenta con acceso a GitHub Copilot y la extensión MSSQL para conectarse a bases de datos SQL Server directamente desde el editor.

## Qué se aprende en este video

- Qué herramientas son necesarias para trabajar con bases de datos en lenguaje natural desde VS Code:
  - **Visual Studio Code** como editor principal.
  - **GitHub Copilot** (cuenta activa) para interactuar con la base de datos mediante lenguaje natural.
  - **Extensión MSSQL** para conectarse y ejecutar consultas sobre SQL Server desde VS Code.
- Cómo acceder al repositorio del curso en GitHub y seguir el material de cada video.

## Prompts trabajados

Este video no incluye prompts. Es una sesión de configuración inicial del entorno.

## Evidencias generadas

Este video no genera archivos de código. El resultado es el entorno listo para trabajar.

## Resultado esperado para el estudiante

Al finalizar este video, el estudiante tiene instalado y configurado VS Code con GitHub Copilot y la extensión MSSQL, y sabe dónde encontrar el repositorio del curso para seguir los ejercicios.
