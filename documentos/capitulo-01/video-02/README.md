# Video 01-02 — Un breve recorrido por VS Code

## Objetivo de aprendizaje

Familiarizarse con el entorno de VS Code antes de comenzar a trabajar con bases de datos, identificando las áreas clave: el explorador de archivos, la extensión SQL Server y el panel de chat con GitHub Copilot.

## Qué se aprende en este video

- **Explorador de archivos**: cómo está organizado el espacio de trabajo y dónde se guardan los archivos del curso.
- **Menú lateral**:
  - El **explorador** de archivos y carpetas del proyecto.
  - La **extensión MSSQL** (SQL Server): cómo navegar conexiones, bases de datos, tablas y ejecutar consultas.
- **Panel de chat con GitHub Copilot**: cómo abrir el chat, elegir el modo de agente y comenzar a interactuar en lenguaje natural.

## Prompts trabajados

Este video no incluye prompts. Es un recorrido visual por la interfaz.

## Evidencias generadas

Este video no genera archivos de código. El resultado es el conocimiento de la interfaz para trabajar con fluidez durante el resto del curso.

## Resultado esperado para el estudiante

Al finalizar este video, el estudiante se siente cómodo navegando por VS Code: sabe dónde están sus archivos, cómo usar la extensión MSSQL y cómo abrir el chat de GitHub Copilot para empezar a gestionar bases de datos en lenguaje natural.
