# Video 02-01

## Objetivo de aprendizaje
Crear una base de datos en SQL Server usando lenguaje natural, sin pedir ni revisar scripts en esta etapa.

## Que se aprende en este video
- Redactar un prompt claro para crear una base de datos.
- Ejecutar la accion directamente desde lenguaje natural.
- Comprender que en este punto del curso se prioriza el resultado directo.

## Prompt trabajado
Crea una base de datos llamada GestorDocumentalDb en SQL Server.

## Evidencia generada
- Base de datos creada directamente en SQL Server mediante lenguaje natural.

## Resultado esperado
El sistema crea la base de datos GestorDocumentalDb de forma directa a partir del prompt.

## Criterio de logro
El estudiante puede crear una base de datos con lenguaje natural sin depender de scripts en este primer acercamiento.

## Nota didactica
La revision y solicitud de scripts se abordara en videos posteriores como parte de buenas practicas.
