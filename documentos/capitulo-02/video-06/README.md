# Video 02-06

## Objetivo de aprendizaje
Redactar correctamente un prompt que solicite la construcción completa de una base de datos relacional para cursos favoritos de LinkedIn Learning.

## Qué se aprende en este video
- Enfocar el desafío en la redacción del prompt, no en la ejecución directa del SQL.
- Definir con claridad los requisitos que la IA debe cumplir al generar el script.
- Consolidar la evidencia del reto en un único archivo dentro de codigo.

## Prompt trabajado
Se realizó el desafío planteado en codigo/desafio.md, centrado en redactar el prompt maestro que pide modelo relacional, claves, relaciones, datos de prueba y consultas de validación.

## Evidencias generadas
- codigo/desafio.md

## Resultado esperado
El estudiante finaliza el video con una formulación precisa del desafío y un prompt listo para generar un script SQL completo y validable.
