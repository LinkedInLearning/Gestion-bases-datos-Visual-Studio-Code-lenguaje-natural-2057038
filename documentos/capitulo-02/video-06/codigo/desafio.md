# Desafío: Redacta el prompt para crear una base de datos de cursos favoritos

## Contexto
En este desafío aplicarás lo aprendido en el capítulo sobre diseño y validación de bases de datos relacionales.

## Objetivo
Redactar un prompt claro, completo y bien estructurado que le pida a una IA diseñar y construir una base de datos para gestionar tus cursos favoritos de LinkedIn Learning.

## Aclaración clave
El desafío no consiste en escribir directamente el script SQL final.
El desafío consiste en crear el prompt que solicite todo lo necesario para que ese script SQL se genere correctamente.

## Requisitos del desafío
1. El prompt debe pedir un modelo relacional coherente con el problema.
2. El prompt debe solicitar tablas con claves primarias y foráneas.
3. El prompt debe exigir relaciones entre entidades (por ejemplo: cursos, autores, categorías, progreso, reseñas o favoritos).
4. El prompt debe incluir la carga de datos de prueba suficientes.
5. El prompt debe requerir consultas de validación para comprobar que la solución funciona correctamente.

## Entregable principal
Crea un prompt detallado que permita generar un script SQL completo, organizado por secciones y con una breve explicación de cada bloque.

## Estructura sugerida del script SQL
- Creación de base de datos.
- Creación de tablas y restricciones.
- Inserción de datos de prueba.
- Consultas de validación.
- (Opcional) vistas o consultas de reporte.

## Resultado esperado
Un prompt de alta calidad que, al ejecutarse en una IA, produzca un script SQL claro, ejecutable y bien estructurado para validar la base de datos propuesta.
