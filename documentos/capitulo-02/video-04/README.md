# Video 02-04

## Objetivo de aprendizaje
Mostrar como ejecutar scripts SQL de forma total o parcial, aprovechando los resultados obtenidos en cada paso.

## Que aprende el estudiante
- Traducir un requerimiento funcional en instrucciones SQL claras.
- Revisar sintaxis, supuestos y riesgos antes de ejecutar cambios manualmente.

## Prompt trabajado
Crea vistas SQL para consulta de documentos, estados y actividad reciente; muestra cada script de vista antes de ejecutarlo para validarlo; documenta cada vista indicando su proposito y tablas implicadas.

## Resultado esperado
El estudiante finaliza el video con vistas reutilizables para consulta y monitoreo, validadas con consultas de prueba antes de su ejecucion manual en SQL Server.
