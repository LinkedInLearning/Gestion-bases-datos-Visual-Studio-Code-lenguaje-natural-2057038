# Video 02-03

## Objetivo de aprendizaje
Solicitar y obtener un script SQL de datos de ejemplo para GestorDocumentalDb, priorizando la revision del codigo antes de su ejecucion manual.

## Que aprende el estudiante
- Diseñar un script de carga de datos de prueba alineado con un esquema relacional real.
- Pedir scripts en lenguaje natural y validarlos antes de ejecutarlos manualmente en SQL Server.

## Prompt trabajado
Crea un script de SQL para llenar la base de datos GestorDocumentalDb con datos de ejemplo, con el fin de comprobar la validez de la misma.

## Evidencias generadas
- codigo/02-datos-ejemplo-gestor-documental.sql
- prompts/01-carga-datos-ejemplo-gestor-documental.md
- Script SQL listo para ejecucion manual y para comprobar conteos, versiones actuales y permisos por documento.

## Resultado esperado
El estudiante obtiene un script reutilizable que puede revisar, ajustar y ejecutar manualmente para validar reglas de integridad antes de continuar con escenarios funcionales.

## Criterio de logro
El estudiante demuestra que sabe solicitar scripts SQL correctos, revisarlos y ejecutarlos manualmente sin romper restricciones del esquema.
