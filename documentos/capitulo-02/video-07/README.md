# Video 02-07

## Objetivo de aprendizaje
Generar y revisar un script SQL Server completo, a partir de un prompt bien redactado, para modelar una base de datos relacional de cursos favoritos de LinkedIn Learning.

## Qué se aprende en este video
- Transformar un prompt en un script SQL estructurado por secciones y listo para revisión.
- Definir tablas, claves primarias y foráneas, restricciones e índices en un modelo relacional coherente.
- Incorporar datos de ejemplo y consultas de validación para comprobar integridad y consistencia.
- Dejar una evidencia reutilizable para que el estudiante ejecute manualmente el script en SQL Server.

## Prompt trabajado
Se trabajó un prompt orientado a generar la base de datos CursosFavoritosLL con creación de base, limpieza de objetos previos, tablas, relaciones, restricciones, índices, datos de ejemplo, consultas útiles y validaciones finales.

## Evidencias generadas
- codigo/01-cursos-favoritosll.sql
- prompts/01-genera-script-sql-cursos-favoritos-linkedin-learning.md

## Resultado esperado
El estudiante finaliza el video con un script SQL completo y organizado para crear, poblar y validar la base de datos CursosFavoritosLL, listo para ejecutarse manualmente en SQL Server como práctica guiada.
