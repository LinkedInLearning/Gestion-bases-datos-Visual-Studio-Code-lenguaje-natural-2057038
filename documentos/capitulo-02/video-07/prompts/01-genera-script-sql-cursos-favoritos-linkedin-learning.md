# Prompt 01 - genera-script-sql-cursos-favoritos-linkedin-learning

Actúa como arquitecto y desarrollador SQL experto en SQL Server.
Genera un único script SQL, listo para ejecutar de principio a fin, para una base de datos llamada CursosFavoritosLL que gestione mis cursos favoritos de LinkedIn Learning.
Incluye estas entidades y relaciones mínimas: cursos, autores, categorías, plataforma, progreso y valoraciones; define correctamente PK, FK y la relación muchos a muchos entre cursos y categorías.
Aplica restricciones de integridad: NOT NULL, UNIQUE y CHECK donde corresponda, e índices iniciales en campos de búsqueda frecuente.
Inserta datos de ejemplo realistas con mínimos obligatorios: 8 cursos, 4 autores, 5 categorías y registros relacionados en todas las tablas.
Incluye consultas útiles: cursos pendientes, cursos completados, mejor valorados, progreso medio por categoría y cursos por autor.
Añade una sección final de validaciones con conteos por tabla y comprobaciones básicas de integridad referencial.
Entrega el resultado ordenado por secciones con comentarios SQL breves antes de cada bloque.
Devuelve solo SQL, sin texto fuera de comentarios SQL.
