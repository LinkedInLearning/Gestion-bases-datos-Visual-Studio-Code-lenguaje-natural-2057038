# Video 03-06

## Objetivo de aprendizaje
Resolver correctamente el desafío del video anterior mediante la escritura de prompts efectivos y aprender a crear prompts con apoyo de IA para obtener entregables técnicos de calidad.

## Qué se aprende en este video
En este video el estudiante aprende a:

- Diseñar prompts claros, precisos y orientados a resultados.
- Guiar a la IA para producir un análisis técnico útil de una base de datos.
- Solicitar un PRD priorizado por impacto y riesgo con estructura controlada.
- Crear prompts de seguimiento para controlar ejecución y avance del plan.

## Prompt trabajado
1. Crea un prompt en español para que una IA analice la base de datos CursosFavoritosLL y luego genere un PRD de mejoras priorizadas por impacto y riesgo, con exactamente dos secciones: “Análisis técnico” y “PRD”.

2. Actúa como arquitecto de datos senior y product manager técnico.
Vamos a trabajar con la base de datos CursosFavoritosLL. Quiero que generes dos entregables en español claro y preciso.

Objetivo:
- Entender la estructura actual.
- Detectar riesgos y oportunidades.
- Proponer un PRD de mejoras priorizado por impacto y riesgo.

Entrega exactamente estas dos secciones:

1. Análisis técnico de la base de datos
- Resumen del dominio de negocio.
- Tablas principales, relaciones y cardinalidades.
- Evaluación de integridad (PK, FK, nulabilidad, consistencia).
- Riesgos de rendimiento, mantenibilidad y escalabilidad.
- Oportunidades de mejora concretas y justificadas.

2. PRD de mejoras priorizadas
- Resumen ejecutivo, objetivos y alcance.
- Lista de mejoras con prioridad alta/media/baja.
- Para cada mejora: impacto, esfuerzo, riesgo y dependencias.
- Cambios propuestos en esquema, vistas y procedimientos.
- Plan por fases (quick wins, estructural, optimización).
- Criterios de aceptación, métricas de éxito y mitigaciones.

Reglas:
- Si falta contexto, declara supuestos y continúa.
- Evita recomendaciones genéricas.
- Usa tablas cuando ayuden a comparar prioridades.

3. Crea una lista de verificación para poder seguir el estado del PRD.
