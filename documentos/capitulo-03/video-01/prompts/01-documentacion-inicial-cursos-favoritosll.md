# Prompt: Documentación inicial de CursosFavoritosLL

Analiza la base de datos CursosFavoritosLL y genera una documentación inicial clara con tablas, relaciones, vistas, procedimientos y objetivos funcionales; incluye un resumen ejecutivo para un DBA que se incorpora al proyecto; identifica dudas o huecos de contexto que necesitemos resolver antes de hacer cambios.

## Contexto
La base de datos CursosFavoritosLL gestiona un catálogo de cursos favoritos de LinkedIn Learning y Microsoft Learn con datos sobre plataformas, autores, categorías, progreso de aprendizaje y valoraciones.

## Requisitos de salida
1. **Resumen ejecutivo**: Estado actual, cantidad de datos, indicadores clave para un DBA nuevo
2. **Modelo lógico documentado**: Descripción de cada tabla con propósito, campos y restricciones
3. **Análisis de relaciones**: Cardinalidades y evidencia de datos
4. **Índices activos**: Listado y justificación
5. **Dudas y decisiones pendientes**: Cambios que podrían impactar el esquema
6. **Recomendaciones**: Próximos pasos para evolución (vistas, procedimientos)
7. **Verificación de integridad**: Confirmación de que los datos son coherentes
