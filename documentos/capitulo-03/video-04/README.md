# Video 03-04

## Objetivo de aprendizaje
Aprender a reactivar un PRD para retomar proyectos ya iniciados, traduciendo el alcance pendiente en ejecución técnica concreta y verificable en SQL Server.

## Qué se aprende
En este video el estudiante aprende a:

- Reactivar un PRD existente para continuar un proyecto sin perder contexto funcional ni técnico.
- Diseñar procedimientos almacenados con validaciones de dominio para alta de cursos.
- Controlar coherencia de estado y porcentaje en el progreso de aprendizaje.
- Registrar valoraciones con reglas de elegibilidad y prevención de duplicados.
- Usar bloques TRY/CATCH con transacciones para garantizar consistencia.
- Validar casos exitosos y fallidos con evidencia de rollback.
- Ejecutar pruebas integradas de extremo a extremo en una base real.

## Prompt trabajado
Se trabajó sobre el PRD v2 para reactivar el proyecto, completar MP1 (procedimientos almacenados core), ejecutar pruebas integradas y dejar trazabilidad del avance en checklist técnico.
