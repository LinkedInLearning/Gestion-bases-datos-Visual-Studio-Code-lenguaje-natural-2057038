# Video 03-05: Desafío - Entender y gestionar tu base de datos

## Objetivo de aprendizaje

Aplicar lo aprendido en los videos 03-01 y 03-02 para analizar una base de datos existente y convertir ese diagnóstico en un PRD priorizado y accionable.

## Qué se aprende en este video

Al completar este desafío, el estudiante practica cómo:
- Documentar de forma efectiva una base de datos existente.
- Analizar estructura, relaciones, integridad y oportunidades de mejora.
- Transformar hallazgos técnicos en decisiones priorizadas de producto y arquitectura.
- Redactar un PRD con enfoque de impacto, riesgo y plan de implementación.

## Resultado esperado para el estudiante

El estudiante termina con:
1. Un análisis técnico completo del modelo descargado.
2. Un PRD de mejoras con prioridades por impacto y riesgo.
3. Una base documental clara para ejecutar mejoras en siguientes iteraciones.
