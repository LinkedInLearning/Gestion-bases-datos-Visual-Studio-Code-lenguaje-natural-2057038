# Desafío: Entender y gestionar tu base de datos

## Objetivo general

Aprenderás a gestionar de manera eficiente una base de datos existente generando la documentación necesaria para su comprensión integral.

## Contexto

En los capítulos anteriores obtuviste un modelo de base de datos descargable. Ahora es tu turno de aplicar los conocimientos adquiridos para documentar y planificar mejoras en esa base de datos.

## ¿Qué debes hacer?

### Paso 1: Crear un prompt para análisis de la base de datos

Utilizando la técnica aprendida en el video 03-01, deberás crear un prompt que genere un **análisis completo** de tu base de datos. Este análisis debe incluir:

- Descripción de todas las tablas y sus relaciones
- Identificación de patrones de diseño
- Análisis de la integridad referencial
- Casos de uso principales
- Oportunidades de optimización

**Consejo:** Reutiliza el enfoque que empleamos en 03-01 para estructurar preguntas claras y específicas que te entreguen un análisis profundo de tu modelo.

### Paso 2: Generar un PRD de mejoras

Siguiendo el flujo que realizamos en el video 03-02, deberás elaborar un **Plan de Requisitos y Mejoras (PRD)** basado en el análisis anterior. Este PRD debe:

- Priorizar mejoras según impacto y riesgo
- Definir cambios en el esquema si aplica
- Proponer nuevas vistas o procedimientos
- Establecer un plan de implementación ordenado
- Incluir criterios de validación

**Consejo:** Aplica la misma estructura de PRD que vimos en 03-02, adaptada a los hallazgos de tu análisis.

## Entregables

Al completar este desafío, tendrás:

1. Un prompt bien estructurado para análisis de BD
2. Un análisis completo de tu base de datos
3. Un PRD priorizado con mejoras concretas
4. Documentación lista para implementar cambios

## Criterios de éxito

- El análisis identifica claramente la estructura y relaciones de la BD
- El PRD incluye mejoras priorizadas según impacto y riesgo
- Ambos documentos son revisables y pueden servir como base para desarrollo futuro
- Tu prompt es reutilizable para otras bases de datos
