# Video 04-02

## Objetivo de aprendizaje
Consultar cursos completados desde SQL Server y comprender por qué una consulta está escrita de una forma específica, evaluando variantes equivalentes.

## Qué se aprende en este video
En este video el estudiante aprende a:

- Construir una consulta con SELECT, FROM, INNER JOIN, WHERE y ORDER BY para resolver una pregunta de negocio.
- Identificar la tabla correcta para filtrar el estado de avance de los cursos.
- Explicar cada cláusula SQL de forma razonada, no solo sintáctica.
- Comparar alternativas de filtrado para definir cuándo un curso está completado.
- Proponer retos de práctica para reforzar criterios de JOIN, filtros y ordenamiento.

## Prompt trabajado
Prompts trabajados:

- Dame los cursos completados de la base de datos CursosFavoritosLL.
- Explícame el SQL generado paso a paso para aprender: qué hace cada cláusula (SELECT, FROM, WHERE, ORDER BY), por qué está escrita así y qué alternativas tendría.
- Dame 5 desafíos para que pueda asimilar los conceptos que me has mostrado.

## Resultado esperado para el estudiante
Al finalizar este video, el estudiante puede recuperar cursos completados con una consulta bien estructurada, justificar técnicamente cada cláusula utilizada y diseñar ejercicios de práctica para consolidar lo aprendido.
